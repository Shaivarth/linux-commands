# pwd

Shows the current working directory.

```bash
pwd
```

Example Output:

```bash
/Users/shaivarth/developer
```

---

# ls

Lists files and folders in the current directory.

```bash
ls
```

---

# ls -la

Shows detailed list including hidden files.

```bash
ls -la
```

---

# cd

Changes directory.

## Move into a folder

```bash
cd backend
```

## Move back one folder

```bash
cd ..
```

## Go to home directory

```bash
cd ~
```

---

# mkdir

Creates a new folder.

```bash
mkdir linux-notes
```

---

# touch

Creates a new file.

```bash
touch notes.md
```

---

# cp

Copies files.

```bash
cp notes.md backup.md
```

---

# mv

Moves or renames files.

## Rename file

```bash
mv old.txt new.txt
```

## Move file to another folder

```bash
mv notes.md docs/
```

---

# rm

Deletes files.

```bash
rm notes.md
```

---

# rm -r

Deletes folders recursively.

```bash
rm -r test-folder
```

---

# clear

Clears terminal screen.

```bash
clear
```

Shortcut:

```bash
Ctrl + L
```

---

# cat

Displays file content.

```bash
cat notes.md
```

---

# Helpful Symbols

| Symbol | Meaning           |
| ------ | ----------------- |
| `~`    | Home directory    |
| `.`    | Current directory |
| `..`   | Parent directory  |
| `/`    | Root directory    |

---
